Aplikasi Komputasi Awan

Aplikasi komputasi awan adalah aplikasi yang dijalankan atau disimpan di server cloud (awan), bukan di komputer lokal. Contoh: Google Drive, Zoom, Gmail, atau website yang di-host di AWS.

Model layanan: IaaS (Infrastructure as a Service), PaaS (Platform as a Service) dan SaaS (Software as a Service)

Our Programs

📚

Infinity Free

InfinityFree adalah penyedia layanan hosting web gratis yang sudah cukup populer di kalangan pelajar, pemula, dan developer yang ingin belajar membuat website tanpa mengeluarkan uang.

🔬

Website Builder

Platform yang memungkinkan kamu membuat website dengan mudah, tanpa perlu menulis kode (coding). Biasanya berbentuk tampilan visual (drag-and-drop), seperti menyusun elemen (teks, gambar, tombol) di layar.

🎨

MySQL PHPmyAdmin

MySQL adalah sistem manajemen basis data (database management system) yang digunakan untuk menyimpan dan mengelola data.

⚽

Github

GitHub adalah platform online untuk menyimpan dan mengelola kode program, menggunakan sistem kontrol versi yang disebut Git.

🌍

Netlify

Netlify adalah platform hosting web gratis (dan berbayar) yang memungkinkan kamu untuk: Mendeploy (meng-online-kan) website statis atau frontend modern Tanpa perlu menyewa server atau konfigurasi rumit

💼

Kodular dan Firebase

Keduanya adalah alat yang sering dipakai bareng untuk membuat aplikasi Android tanpa coding (Kodular) dan mengelola data online (Firebase). Cocok banget untuk pelajar, mahasiswa, atau pemula yang ingin membuat aplikasi mobile.
